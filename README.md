A beautiful, minimalist life tracker app built with Flutter. Track your sleep, mood, nutrition, and habits in one clean and calming experience.

## Features

- 🛌 **Sleep Tracking** — Log bedtime, wake time, and sleep quality with weekly charts.
- 😊 **Mood Tracking** — Daily mood logging with an emoji picker and a 4-week calendar view.
- 🥗 **Nutrition** — Track meals and water intake with a simple, user-friendly interface.
- ✅ **Habits** — Create custom habits and monitor your consistency over time.
- 📊 **Dashboard** — View your daily wellness summary at a glance.

## Design

- Minimalist, nature-inspired color palette with clean and calming visuals.
- Clean typography for a polished look.
- Smooth animations and transitions.
- Card-based UI with soft shadows.
- Simple and intuitive layout for daily use.

## Tech Stack

- **Flutter** with Material 3
- **Provider** for state management
- **SQLite (sqflite)** for offline local storage
- **fl_chart** for charts and insights
- **Google Fonts** for typography

## Getting Started

### Prerequisites

1. Install [Flutter SDK](https://docs.flutter.dev/get-started/install)
2. Add Flutter to your system PATH

### Setup

```bash
cd MindBloom
flutter pub get
flutter run
```
